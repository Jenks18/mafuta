## üé® How to Use These

### For Google OAuth Consent Screen
1. Open `mafutapass-icon-only.svg` in your browser
2. Take a screenshot or convert to PNG (512x512px)
3. Upload to Google Cloud Console ‚Üí OAuth consent screen ‚Üí App logo

### For Canva Editing
1. Go to https://www.canva.com/
2. Create new design (512x512px)
3. Upload any of the SVG files
4. Edit colors, text, or design elements
5. Export as PNG when done

### Quick Preview
